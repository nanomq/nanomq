# QUIC Features Overview

This section summarizes QUIC-related capabilities in NanoMQ, including:

- Using QUIC as the transport layer for MQTT bridges between edge and cloud;
- Improving stability and latency in weak, high-latency or lossy networks;
- Seamlessly switching between QUIC and traditional TCP via hybrid bridging.
