# 配置说明

## 简介

NanoMQ 的配置文件通常以 .conf 作为后缀名，你可以在 etc 目录找到这些配置文件，主要配置文件包括：

| 配置文件                | 说明                |
| ----------------------- | ------------------- |
| etc/nanomq.conf         | NanoMQ 配置文件     |
| etc/nanomq_gateway.conf | NanoMQ 网关配置文件 |
