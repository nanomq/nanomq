## v1.1.0.beta.1
Release Date: 2025-09-05

### Enhancements
- Added EMQX Edge packages for Windows.
- Added TLS support for Windows EMQX Edge.
- Added Sqlite and Rule Engine support for Windows EMQX Edge.
- Updated Dashboard to version 0.0.4-beta.2.
- Added switch to enable or disable ACL rules.
- Added switch to enable or disable bridge cache.
- Updated default maximum packet size to protocol defined value.

### Fixes
- Fixed the statistics of bytes sent and received.
- Fixed the the error that build-in mqtt client keep reconnecting when auth.http is enabled.
